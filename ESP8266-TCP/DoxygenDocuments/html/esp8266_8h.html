◆ Command_Process()

Status Command_Process ( char **  commandArray,
char **  responseArray,
uint8_t  numberOfCommands 
)

This function handles AT commands and waits the messages expected from the ESP8266.

Parameters
commandArrayis a two dimensional string array containing AT commands.
responseArrayis a two dimensional string array containing response messages expected from the ESP8266.
numberOfCommandsparameter represents the number of commands passed as a parameter in the commandArray.
Return values
STATUS_OK:It returns when ESP send the OK message.
STATUS_ERROR:It returns when ESP send the ERROR message or number of commands is equal zero.
TIMEOUT_ERROR:It returns TIMEOUT_ERROR when timeout occurs. Default timeout is 5000 ms.
IDLE:If there is not a string in the buffer and timeout does not occur yet, it returns IDLE.
